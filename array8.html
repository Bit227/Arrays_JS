Ejercicio con Arrays 8

Dado un array que contiene["azul", "amarillo", "rojo", "verde", "café", "rosa"] determinar si un color introducido por el usuario a través de un input y tras pulsar un botón, se encuentra dentro de un array o no.

Solución propuesta:

var colores = ["azul", "amarillo", "rojo", "verde", "café", "rosa"];
var colorUsuario = prompt ('Escribe un color para ver si está en la lista');
var respuestaUsuario = colores.includes(colorUsuario);
console.log(respuestaUsuario);

if (respuestaUsuario == true) {
window.alert("¡Muy bien! El color está en la lista");
} else {
window.alert("¡Qué pena! El color no está en la lista");
}
